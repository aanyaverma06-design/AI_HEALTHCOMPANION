# 🩺 AI Health Companion

AI Health Companion is a full-stack AI-powered web application that helps users understand their medical reports.

Users can upload PDF reports, receive AI-generated health summaries, identify abnormal values, and ask follow-up questions using an AI chatbot powered by Google Gemini.

---

## ✨ Features

- 📄 Upload medical reports (PDF)
- 🤖 AI-generated report analysis
- ❤️ Health score & risk assessment
- 🔍 Detect abnormal values
- 💬 Chat with your report using AI
- ⚡ FastAPI backend
- ⚛️ React frontend
- ☁️ Deployed on Render & Vercel

## Installation

Clone the repository

```bash
git clone https://github.com/aanyaverma06-design/AI_HEALTHCOMPANION.git
```

Install frontend

```bash
cd frontend
npm install
npm run dev
```

Install backend

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```

---

## Environment Variables

Create a `.env`

```
GEMINI_API_KEY=your_api_key
```
